# GTC Garage Shopify Theme

GTC Garage is a Shopify-based automotive content-commerce project.

The brand combines an online shop with automotive content, garage projects, product testing, and “Shop the Build” pages. Shopify is used as the commerce backend for products, collections, checkout, payments, shipping, tax settings, customer accounts, policies, orders, and supplier/dropshipping integrations.

This repository contains the Shopify theme code for the GTC Garage storefront.

---

## Critical project rule

This is a Shopify theme project.

Do not convert this project into a normal React, Next.js, Vue, Laravel, WordPress, or custom ecommerce project.

Do not change the required Shopify theme folder structure.

Do not remove, rename, or restructure Shopify core theme directories unless there is a very specific Shopify-compatible reason.
